Detailed Description

template<class CONTROLLED_SYSTEM>
class ct::core::ControlSimulator< CONTROLLED_SYSTEM >

A class for simulating controlled systems in a general way.

This runs two threads - one that integrates the controlled system by applying the defined control, and one that updates the control if needed.

Template Parameters
CONTROLLED_SYSTEMthe controlled system that we wish to simulate
