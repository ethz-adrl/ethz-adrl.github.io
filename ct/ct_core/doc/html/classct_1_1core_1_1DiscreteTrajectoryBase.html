Detailed Description

template<class T, class Alloc = Eigen::aligned_allocator<T>, typename SCALAR = double>
class ct::core::DiscreteTrajectoryBase< T, Alloc, SCALAR >

A discrete, timed trajectory with interpolation.

This implements a trajectory based on discrete data points which can be evenly or unevenly distributed in time with time stamps associated with each data point. A basic interpolation strategy can evaluate the trajectory also in between data points.

An example how to use different features of the discrete trajectory can be found in unit test DiscreteTrajectoryTest.cpp

Template Parameters
Ttype of each point of the trajectory
Allocallocator for trajectory points
Examples:
DiscreteTrajectoryTest.cpp.
