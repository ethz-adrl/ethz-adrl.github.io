Detailed Description

template<class T, typename SCALAR>
class ct::core::TrajectoryBase< T, SCALAR >

Basic interface class for a trajectory.

This defines a general interface for a trajectory, independent of its implementation. Possible implementations are a DiscreteTrajectoryBase or also splines, basis functions etc.
