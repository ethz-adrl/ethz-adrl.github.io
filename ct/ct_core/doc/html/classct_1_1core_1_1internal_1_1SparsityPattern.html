const CppAD::vector<size_t>& ct::core::internal::SparsityPattern::row ( ) const

returns the row indices of a row-column sparsity representation

In a row-column representation, only the non-zero entries get saved. This method returns the row indices of the row-column representation. Since there can be several non-zero entries in the same row, the same row index can appear repeatedly in this vector.
