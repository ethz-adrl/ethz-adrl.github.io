Detailed Description

template<typename SCALAR>
class ct::core::tpl::ActivationBase< SCALAR >

Base class for all sort of (scalar) activation functions, e.g. time activations. Activations can be used to periodically scale costs, constraints, or other, arbitrary functions. All activations are scalars.
