Detailed Description

template<size_t STATE_DIM, size_t CONTROL_DIM, typename SCALAR = double>
class ct::optcon::CostFunctionQuadraticSimple< STATE_DIM, CONTROL_DIM, SCALAR >

A simple quadratic cost function.

A simple, purely-quadratic cost function of the form where indicate deviations from a nominal (desired) state and control

Examples:
oscDMSTest.cpp, oscDMSTestAllVariants.cpp, switched_continuous_optcon.cpp, and SymplecticTest.h.
