Detailed Description

template<size_t STATE_DIM, size_t CONTROL_DIM, typename SCALAR = double>
class ct::optcon::SwitchedLinearConstraintContainer< STATE_DIM, CONTROL_DIM, SCALAR >

A container for switching linear constraint containers.

  • The LinearConstraintBase Class is the base class for defining the non-linear optimization constraints. This class switches between several of them based on a predefined sequence.
Template Parameters
STATE_DIMDimension of the state vector
CONTROL_DIMDimension of the control input vector
Examples:
switched_continuous_optcon.cpp.
