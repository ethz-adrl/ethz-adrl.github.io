Detailed Description

template<class KINEMATICS, bool FB, size_t STATE_DIM, size_t CONTROL_DIM>
class ct::rbd::TermTaskspacePosition< KINEMATICS, FB, STATE_DIM, CONTROL_DIM >

A costfunction term that defines a cost on a taskspace position.

Warning
This term does not consider orientations. For a full pose cost function, see TermTaskspacePose.h

This cost function adds a quadratic penalty on the position offset of an endeffector to a desired position
